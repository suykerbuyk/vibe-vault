# Dashboards

These dashboards use [Dataview](https://github.com/blacksmithgu/obsidian-dataview) queries to provide live overviews of your vault content.

## Prerequisites

Install the [Dataview](https://github.com/blacksmithgu/obsidian-dataview) community plugin in Obsidian. Enable JavaScript queries in Dataview settings.

## Available Dashboards

| Dashboard | Shows |
|-----------|-------|
| `sessions.md` | All session notes, sorted by date |
| `decisions.md` | Architectural and design decisions |
| `action-items.md` | Open action items from sessions |
| `weekly-digest.md` | Week-by-week summary of activity |
| `by-project.md` | Sessions and knowledge grouped by project |
| `analytics.md` | Token usage, friction trends, tool usage, and project analytics |
