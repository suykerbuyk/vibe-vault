# Projects

Each subdirectory is a project. Structure:

```
Projects/{project}/
├── agentctx/          ← AI context package (created by vv context init)
│   ├── CLAUDE.md      ← Behavioral rules and file pointers
│   ├── resume.md      ← Project state, architecture, decisions
│   ├── iterations.md  ← Iteration narratives and history
│   ├── commands/      ← Slash commands (/restart, /wrap)
│   └── tasks/         ← Active and completed tasks
├── sessions/          ← Auto-generated session notes (YYYY-MM-DD-NN.md)
└── history.md         ← Auto-generated project context (vv index)
```

The `agentctx/` directory contains all AI context for a project — behavioral
rules, project state, commands, and tasks. The repo-side CLAUDE.md is a thin
pointer to `agentctx/workflow.md`. Slash commands are symlinked from the repo's
`.claude/commands/` to `agentctx/commands/`.

Session notes and history are auto-generated by `vv` — do not edit manually.
